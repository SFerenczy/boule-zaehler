# Boule Zähler

Einfacher Zähler für Boule-Spiele. Erfasst die Erfolgsquote beim **Legen** und **Schießen** für zwei Teams.

## Funktionen

- Zwei Teams mit je zwei Kategorien (Legen / Schießen)
- Erfolg- und Fehlschlag-Buttons pro Kategorie
- Anzeige der Erfolgsquote in Prozent
- Schutz vor versehentlichem Neuladen (Session Storage)
- Spiel zurücksetzen per Knopfdruck
